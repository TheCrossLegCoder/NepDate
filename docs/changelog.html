Release History

Changelog

All notable changes to NepDate are documented here. Versions follow semantic versioning.

v2.0.7

Latest May 2026

Corrected English transliterations of Tithi names and calendar events.

v2.0.6

November 2025

Resolved a NuGet publish conflict from v2.0.5 and stabilized the CI release workflow.

v2.0.5

October 2025

Major release adding calendar metadata, date ranges, fiscal year helpers, smart parsing, bulk conversion, serialization support, and a significant performance overhaul.

v2.0.0

2024

Breaking release restructuring NepaliDate into a readonly partial struct with reorganized namespaces and a unified parse API.

v1.x

2023

Initial public releases establishing the core NepaliDate type with bidirectional BS ↔ AD conversion, parsing, and arithmetic.
